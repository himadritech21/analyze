Welcome to the Data Processing Pipeline

This project demonstrates a robust data processing workflow using Python, Pandas, and GitHub Actions.

Project Overview

The core of this project is the execute.py script, designed to read and process data from an Excel file (data.xlsx). It performs calculations and aggregations, outputting the results as a JSON file.

A key feature is the continuous integration (CI) pipeline set up with GitHub Actions. On every push, the workflow:

  • Lints the Python code using ruff.
  • Executes execute.py to generate result.json.
  • Publishes the generated result.json artifact via GitHub Pages.
